## Abstract

```
This project develops a system of autonomous AI agents that collaborate to simulate 
a real-world software development team. Each agent performs a specific role such 
as planning, coding, and testing.  

The system demonstrates how multiple AI agents can coordinate tasks, 
improving automation and efficiency in complex workflows.
```

---

## Introduction

```
AI systems are evolving from single-task automation to collaborative environments.  
This project explores how multiple AI agents can work together to achieve 
complex goals.
```

## Methodology

```

A central system assigns tasks and manages communication.  

Agents collaborate iteratively to complete workflows efficiently.
```
